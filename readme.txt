엑셀을 읽어서 바이너리데이터와 c#코드를 만들어 준다.
엑셀관련 플러그인은 https://github.com/mikito/unity-excel-importer 에서 가져왔다.
파일 여러개 선택을 위해서 https://github.com/gkngkc/UnityStandaloneFileBrowser를 가져다 썼다.
프로젝트는 uinty 2020.3.33f1버전으로 테스트를 했다.
처음 프로젝트를 읽어 들이면 "Assembly 'Library/ScriptAssemblies/Unity.CollabProxy.Editor.dll' will not be loaded due to errors"와 같은 에러 메세지가 발생한다.
그때는 Menu>>Window >> Package Manager를 열어서 Version Control을 삭제 해주면 사라진다.



혹시 문의 사항이 있을 경우 yakmo94@naver.com으로 메일을 보내주면 확인해보겠습니다.
